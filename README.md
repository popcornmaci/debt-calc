#Tartozás Kalkulátor
####(debt-calc)

##Használat

Ez az alkalmazás lehetőséget biztosít arra, hogy egy vagy több vásárlás alkalmával több ember között fennállható tartozást számítsunk ki.

##Előkövetelmény

-maven 3.x
-jdk 1.8

##Futtatás
project könyvtárából a következő parancsokkal:
-Fordítás:
	$ mvn package
-Futtatás:
	$ java -jar target/debt-calc-jar-with-dependencies.jar
vagy
	$ mvn exec
